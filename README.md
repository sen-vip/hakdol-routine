# 학돌 루틴집 v3.2.6

학교 행정 업무를 월별 타임라인과 담당업무 기준으로 확인하는 학돌루틴 3.2.6 버전입니다.

## 3.2.6 업데이트

- 업무 상세 펼침 영역에서 `해야 할 일`, `확인할 것` 섹션 삭제
- 업무 상세는 업무 설명, 관련 법령, 참고사항, 내 메모 중심으로 간소화
- 전체 레이아웃 최대폭을 조정해 넓은 화면에서 좌우로 과하게 벌어지지 않도록 정리
- 담당업무 필터 버튼을 2열 그리드로 정렬하고 라벨/숫자 간격 보정
- 월별 업무 목록 상단의 보기 토글, 검색창, 정렬 박스 폭을 안정적으로 재배치

## 유지 사항

- 기존 체크/메모/검색/월 선택/필터/정렬/LocalStorage 구조 유지
- 데이터 구조와 업무 목록 내용은 변경하지 않음
