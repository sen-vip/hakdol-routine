# 학돌 루틴집 v3.2.7

학교 행정 업무를 월별 타임라인과 담당업무 기준으로 확인하는 학돌루틴 3.2.7 버전입니다.

## 3.2.7 업데이트

- 한글 검색 입력 중 첫 자음만 남고 조합이 끊기던 문제 수정
- 검색창은 유지하고 업무 결과 목록만 갱신하도록 렌더링 구조 분리
- `compositionstart`·`compositionend`·`isComposing`을 반영해 한글 IME 입력 안정화
- 검색 중 포커스와 커서가 유지되도록 개선
- 검색 결과 목록에만 행 이벤트를 다시 연결해 불필요한 전체 재렌더링 축소

## 유지 사항

- 기존 체크/메모/검색/월 선택/필터/정렬/LocalStorage 구조 유지
- 데이터 구조와 업무 목록 내용 및 디자인은 변경하지 않음
