# 학돌 루틴집 v3.2

학교 행정 업무를 월별 타임라인과 담당업무 기준으로 확인하는 학돌루틴 3.2 버전입니다.

## 3.2 업데이트

- 업무 상세 펼침 영역을 매뉴얼 카드 구조로 정리
- 담당업무 필터 자동 복원 및 현재 보기 문구 반영
- 전체/남은 업무만 보기 토글 추가
- 완료율, 완료 건수, 남은 업무 건수 표시
- 학돌T 업무코치 카드: 현재 월, 담당업무, 완료율, 남은 업무 기준으로 안내 문구 자동 생성

## 유지 사항

- 기존 체크/메모/검색/월 선택/LocalStorage 구조 유지
- 즐겨찾기 및 참고링크 등록 기능은 포함하지 않음
